¿Qué es Yarn?

Yarn es un gestor de paquetes, permite usar y compartir codigo con otros desarrolladores de todo el mundo. El codigo se comparte con algo llamado un paquete (a veces referido como modulo). Un paquete contiene todo el codigo que se comparte, asi como un package.json archivo que describe el paquete.
